Curated Solidity, committed ABIs, and public chain manifests for independent verification.

- `src/` — contract source (mirrors private `apps/contracts/src`)
- `abis/` — generated registry ABI for `@filosign/verify`
- `chains/manifest.json` — public deploy addresses (update on each release)

No deploy scripts or private `definitions/` artifacts.
